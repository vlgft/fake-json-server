# **Fake Json Server**

## Init project

> npm init

set package name, version, entry point,...

> create file db.json

install nodemon for dev enviroment

> npm i --save-dev nodemon

## Install

Install json-server

> npm install -g json-server

Install jsonwebtoken

> npm install jsonwebtoken --save

Install body-parser

> npm i --save body-parser
